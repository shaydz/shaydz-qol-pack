# Shaydz's QoL

A personal Factorio 2.0 dependency mod. Its only purpose is to bundle all of Shaydz's preferred QoL, utility, and RPG mods into a single installable package. Enable this mod and the Factorio mod manager will suggest all the rest.

All mods are **optional** dependencies — you can still toggle any of them off individually after installing.

## Included Mods

- **smart-starter-pack** — Shaydz's own starter items mod

## Installation
Place the `shaydz-qol-pack_1.0.0` folder (or its zip) into your Factorio `mods` directory:
`%APPDATA%\Factorio\mods\`
